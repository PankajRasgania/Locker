Information locker

What is information locker?

Information locker is a tool where you can save AES-128 encrypted critical sensitive information (such as passwords, secrets, etc.) for later use.

How it works?

To add information, click on "Add info" button given on the top options. Enter critical information in "Raw data" field. To encrypt this information a key will be required. This key will enable you to retrieve your information back later. Enter your desired key in "Key" field. There is an additional field of "Label". Every information can be labelled so that you know what that information is about. Labels are not encrypted and will be shown as it is in plain text along with the information. Labels are optional.

Once you have added information to this locker (or made any changes), you can save this html page in your device. To do this, right click and click on "save as". If an earlier version already exists on you device you should overwrite it. All information is saved in this html file. You can store this html file anywhere (could storage, disk, usb, etc.). It can be run in any browser.

Alternatively you can export your data into in a file using the "Export" option and save it anywhere you like. To see the information in the file, visit https://pankajrasgania.in/tools/locker.html or run this html file in a browser, and import your data file from the "Import" option. You can then decrypt you data.

How secure it is?

This application runs entirely on client side and no data is sent to any server. If you doubt this, you can download this html file and run it locally in a browser without internet.

This application uses Advanced Encryption Standard (AES) 128 bit encryption algorithm which is one of the best encryption algorithm at present. This tool is based on the Stanford Javascript Crypto Library (a project by the Stanford Computer Security Lab) which can be accessed here: https://bitwiseshiftleft.github.io/sjcl
